# Chrome DevTools Rules

## 1. Live DOM Inspection
- Proactively leverage CDP commands to check the active page state rather than relying on static source HTML.
- Inspect accessibility tree states, layout spacing, and interactive targets.

## 2. Performance Profiling
- Audit Largest Contentful Paint (LCP) and Interaction to Next Paint (INP) using DevTools trace logs.
- Identify blocking scripts and unoptimized images.

## 3. Error Diagnostics
- Capture Console error logs and network request statuses to diagnose client-side crashes.
