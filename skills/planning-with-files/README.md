# Manus-style File-Driven Project Orchestration

Orchestration rules using persistent markdown files (implementation_plan.md, task.md, walkthrough.md) as agent memory.

## Getting Started
To trigger this skill inside your Claude Code session, use any of the keywords or phrases described in `SKILL.md`.

## Contents
- `SKILL.md`: Main instructions and trigger description for the AI.
- `rules.md`: Comprehensive workspace rules and constraints for this specific domain.
- `references/`: Sub-guidelines, templates, or checklists.
- `scripts/`: Supporting helper scripts.
